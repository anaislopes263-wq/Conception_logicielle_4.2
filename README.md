# Module 1 – Java

## Prérequis

Vérifier la version Java installée :

```bash
java -version
```

> Version utilisée : `openjdk version "26" 2026-03-17`

---

## Installation & Compilation

### 1. Se placer dans le dossier du module

```bash
cd module_1/Module_1_Java
```

### 2. Compiler tous les fichiers Java

```bash
javac *.java
```

> Cela génère les fichiers `.class` nécessaires à l'exécution.

---

## Lancer le programme

```bash
java Main
```

---

## Lancer les tests

```bash
java TestEncapsulation
java TestHeritage
java TestPolymorphisme
```
